# Scientific Computing with Python Projects

This directory contains a collection of projects from the FreeCodeCamp "Scientific Computing with Python" certification. Each project is in its own directory and includes a detailed README with specific requirements and examples.

## Projects

Here is a summary of the projects included:

### 1. Arithmetic Formatter
-   **Description:** A function that arranges arithmetic problems vertically and side-by-side, with an option to display the answers.
-   **Directory:** [Arithmatic formatter](./Arithmatic%20formatter/)

### 2. Budget App
-   **Description:** A `Category` class to manage a budget, including deposits, withdrawals, and transfers. It also includes a function to create a spending chart.
-   **Directory:** [Budget App](./Budget%20App/)

### 3. Polygon Calculator
-   **Description:** `Rectangle` and `Square` classes with methods to calculate area, perimeter, diagonal, and to render the shape as a string of asterisks.
-   **Directory:** [Polygon calculator](./Polygon%20calculator/)

### 4. Probability Calculator
-   **Description:** A `Hat` class and an `experiment` function to simulate drawing balls from a hat and estimate probabilities.
-   **Directory:** [Probability Caclculator](./Probability%20Caclculator/)

### 5. Time Calculator
-   **Description:** A function that adds a duration to a start time and returns the new time, handling AM/PM changes and day rollovers.
-   **Directory:** [Time calculator](./Time%20calculator/)
